IMPORTANT LUAU NOTES:
- USING THE # WHEN FINDING THE SIZE OF A TABLE/ARRAY DOES NOT COUNT INDEX 0 SO THERES AN OFF BY 1 ERROR
- USING THE `FOR ITEM IN TABLE` SYNTAX IGNORES ANYTHING WITH THE INDEX 0
- IF A FUNCTION IS CALLED A LOT THEN ITS WORTH IT TO LOCALIZE SAID FUNCTION
- IF A FUNCTION HAS HIGH TRAFFICE ALWAYS TRY TO PREALLOCATE STORAGE FOR THE TABLE AND FLUSH/REUSE WHEN POSSIBLE
- BUFFERS ARE MORE PERFORMANT THAN TABLES SO TRY TO USE THEM OVER TABLES FOR PERFORMANT FUNCTIONS
